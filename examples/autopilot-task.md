# Пример задачи для автопилота

```text
Проанализируй текущий репозиторий, найди самую маленькую реальную проблему, которую можно подтвердить проверкой, исправь её, запусти релевантные тесты/сборку, очисти generated-файлы и сделай commit через /git/commit.

Правила:
- не выполняй git push;
- используй /git/commit, а не прямой git commit;
- для большого вывода используй capture=file;
- в конце покажи task report.
```
